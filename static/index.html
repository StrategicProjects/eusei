Consultas do SEI
em JSON, simples.

API HTTP que conversa com os Web Services SOAP do SEI e devolve JSON limpo: processos, documentos, andamentos e listas — espelhando o pacote R rsei.

Explorar a documentação Início rápido
consultar processo 
curl -H 'Authorization: Bearer SEU_TOKEN' \
  '…/eusei/v1/procedimento?protocolo=0011108545.000056/2022-49'

{
  "ok": true,
  "dados": {
    "ProcedimentoFormatado": "0011108545.000056/2022-49",
    "Especificacao": "TOMADA DE PREÇOS …",
    "TipoProcedimento": { "Nome": "Licitação…" }
  }
}

Início rápido

1 · Autenticação

Toda rota /v1 exige o cabeçalho:

Authorization: Bearer <token>
2 · Protocolo na query

Passe o número por ?protocolo= (a barra sobrevive ao proxy):

/v1/procedimento?protocolo=…/2022-49
3 · Resposta

Sempre JSON. Erros trazem um codigo estável e status HTTP coerente.

{ "ok": true, "dados": … }
